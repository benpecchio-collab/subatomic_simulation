# subatomic simulation
this is a js simulation of the forces between subatomic particles and between more complex systems as atoms. There are multiple funcions to simulate many scenarios. You can rewind time, speed it up, add and remove particles, freeze time and more! Have fun!
LINK: https://benpecchio-collab.github.io/subatomic_simulation/

DISCLAIMER: 
the atomic model is not totally accurate |
the scale is limited to protons, neutrons and electrons, it doesn't reach quarks and doesn't include bosons |
the simulator is in development
